# AddLine
**Description:**
This plugin will add a line in a text file at the desired location (Line number). it is as simple as calling the plugin (in this case, addline) and specifying the file name, line number and text to insert.

AUTHOR:	**Kvc**

# USAGE

<code>Call AddLine [File] [LineNumber] [NewLineContent]</code>

​	Examples: 		AddLine "Demo.txt" 5 "This is a Test"
